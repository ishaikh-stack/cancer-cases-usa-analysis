# Cancer Cases in the USA (2017)

This repository contains the complete data visualization and analysis project for cancer cases in the United States.

Files:
- Data_visualization_Project.ipynb → Main Jupyter Notebook
- cancer2017.csv → Dataset used
- outputs/ → All generated charts in high resolution

Reproducibility:
1. Clone or download this repository.
2. Open the notebook using Jupyter.
3. Install required libraries:
   - pandas
   - numpy
   - matplotlib
   - seaborn
4. Run all cells from top to bottom.
5. All outputs will be recreated automatically.

Data Source:
Kaggle – Cancer Cases in the USA (2017)
