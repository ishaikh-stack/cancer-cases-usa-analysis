# Cancer Cases in the USA (2017)

This repository contains the complete data visualization and analysis project for cancer cases in the United States for the year 2017.

## Project Overview
The project explores:
- Distribution of total cancer cases across U.S. states
- Top 10 and Bottom 10 states by total cancer cases
- Distribution of new cancer cases across cancer site types
- Comparison of cancer deaths between men and women
- Gender-based mortality patterns
- Public health implications

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly (for interactive visualization)

## Files in this Repository

- `Data_visualization_Project-IrumShaikh.ipynb` – Main Jupyter Notebook
- `cancer2017.csv` – Primary dataset (Kaggle)
- `reshaped_cancer_data.csv` – Processed dataset
- `both_table_df.csv` – Combined statistics dataset
- `outputs/` – Saved chart images (PNG)

## How to Reproduce the Analysis

1. Clone or download this repository.
2. Open the Jupyter notebook file:
   `Data_visualization_Project-IrumShaikh.ipynb`
3. Ensure the required libraries are installed.
4. Run the notebook cells from top to bottom.
5. All charts will be displayed and saved in the `outputs` folder.

## Dataset Source
Kaggle – Cancer Cases in the USA (2017)
